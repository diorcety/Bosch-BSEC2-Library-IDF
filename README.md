# ESP-IDF BSEC2 Wrapper

Simple ESP-IDF component wrapping the Bosch BSEC2 library for BME68x sensors.

## Features

* Easy integration with ESP-IDF
* Includes BSEC2 as a submodule
* Auto-selects correct binary for ESP32 targets
* Uses Bosch BME68x driver

## Dependencies

* Bosch BSEC2 Library
* Bosch BME68x Library (IDF version)
